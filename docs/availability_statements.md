# Suggested manuscript text

## Data Availability

The de-identified data underlying this study are not publicly available because they contain sensitive patient information and are subject to institutional and ethical restrictions. Data may be made available to qualified researchers upon reasonable request to the corresponding author, subject to institutional review board or ethics approval and completion of any required data-use agreement.

## Code Availability

The R code and supporting documentation used for data cleaning, statistical analysis, and figure generation are publicly available in the GitHub repository `glaucoma-surgery-trends` and archived in Zenodo ([DOI to be inserted after public release]).
